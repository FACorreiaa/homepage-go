# Hermes Agents get Firecrawl integration upgrade

@mr_r0b0t announces that NousResearch Hermes Agents now integrate with Firecrawl for free web scraping, search, and PDF parsing without an API key. Firecrawl's MCP, CLI, and API endpoints are live for immediate use.

## Relevance

Directly impacts the Hermes Agent ecosystem you operate. Firecrawl provides a zero-setup web scraping and search layer that could enhance Hermes' web-reading capabilities (web_search, web_extract, browser tools) with more robust parsing, PDF handling, and JavaScript-rendered content extraction. The free tier with no API key requirement lowers the barrier for testing integration into Hermes skills or vault ingestion pipelines.

## Notes

- Firecrawl supports: web search, scrape any page, interact with pages, parse PDFs to clean markdown
- Available via MCP, CLI, and REST API
- No API key needed to start; sign up only when scaling
