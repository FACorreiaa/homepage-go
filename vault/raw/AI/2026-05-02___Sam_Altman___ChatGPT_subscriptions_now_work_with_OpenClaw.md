# Sam Altman — ChatGPT subscriptions now work with OpenClaw

## Definition
Sam Altman confirms that OpenClaw (a Claude Code alternative) can now authenticate using a ChatGPT Plus/Pro subscription, allowing users to leverage their existing OpenAI payment to access the tool. This comes after Anthropic disrupted access for some Claude Code users on their subscription.
