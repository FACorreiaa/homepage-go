# LangChain / LangSmith Product Updates — May 2026
Ingested: 2026-05-15T09:08:00Z
Source: https://blog.langchain.dev

## Major Launches

- **LangSmith Engine** — new LLM execution engine built into LangSmith
- **LangChain Labs** — new initiative/program for cutting-edge agent research
- **SmithDB** — dedicated data layer for agent observability
- **LangSmith Context Hub** — centralizes context management across agent runs
- **LangSmith Sandboxes** — now GA (ephemeral agent execution envs)
- **Managed Deep Agents** — fully managed hosted Deep Agents (no self-host headache)
- **Deep Agents v0.6** — improved model tuning + runtime
- **LangSmith LLM Gateway** — runtime governance inside the agent lifecycle
- **Delta Channels** — async pub/sub runtime for long-running agent tasks
