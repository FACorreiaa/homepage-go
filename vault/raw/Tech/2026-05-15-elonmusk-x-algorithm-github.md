# X/Twitter Recommendation Algorithm Published to GitHub

Link: https://github.com/xai-org/x-algorithm

Tweet by Elon Musk (@elonmusk) announcing that the latest X recommendation algorithm has been published to GitHub under the xai-org organization.

## Key Points

- The X (Twitter) feed ranking algorithm is now open-sourced at github.com/xai-org/x-algorithm
- Hosted under xAI's GitHub organization, not Twitter/X's
- Signals continued transparency push by Musk post-acquisition

## Relevance

High relevance if you're studying recommendation systems or ML infrastructure. Unusual for a social platform to open-source core ranking logic. Worth a skim to see signal weighting, engagement scoring, and content ranking structure.
