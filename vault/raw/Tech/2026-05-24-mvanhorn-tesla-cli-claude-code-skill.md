# Tesla CLI / Claude Code Skill / OpenClaw and Hermes Skill

**Matt Van Horn** introducing a Tesla CLI tool built as a Claude Code Skill, OpenClaw skill, and Hermes skill from the @ppressdev (Printing Press).

## Features

- **"Unlock the car"** and **"turn on dog mode"** as one-line commands, callable from your phone or laptop
- **Agent scheduling:** "during winter school days, defrost my car at 7:50 every weekday before school dropoff"
- **Charging cost ledger**
- **Supercharger queue watcher** — pageable from an agent
- **Your signing key stays on YOUR host**

## Background

Fun fact: when Matt got his first ClawdBot, Tesla was one of the first skills he made. But he could only get it to work with his older Tesla. Now that he has the Printing Press, he was able to build what he wanted soup to nuts.
